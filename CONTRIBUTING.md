# Contributing

- **Build:** JDK **11+** (see **`maven.compiler.release`** in **`pom.xml`**) and **Maven 3.6+** (`mvn` on **`PATH`**). This branch tracks WSDL/XSD under **`wsdls/xml/`** in Git; from the repo root **`mvn clean compile`** uses that tree (no project **`settings.xml`** required - **`hi.wsdl.tree.root`** defaults in the POM). **`HI_WSDL_TREE_ROOT`** or **`mvn "-Dhi.wsdl.tree.root=..."`** override the path for parity with other branches. **`wsimport`** merge/clear uses **`gmavenplus-plugin`** with **plugin-scoped** **`groovy`** and **`groovy-ant`** (not shipped in the library JAR). With tests: **`mvn -B -Dgpg.skip=true clean verify`**. Optional faster local **`verify`** without the Javadoc JAR: **`mvn -B -Pdev-javadoc-off -Dgpg.skip=true clean verify`**. Full HI mutual-TLS tests: **`mvn -B -Pintegration -Dgpg.skip=true clean test`**. **`mvn -B -Psample -Dgpg.skip=true -DskipTests=true clean compile`** type-checks **`src/sample/java`**. Optional **`./build.sh`**, **`build.ps1`**, or **`build.bat`**: same **`verify`** (set **`MVN_SETTINGS`** if you must pass **`-s`**). Append **`shaded`** for **`mvn -Pfat-jar`** (classifier **`all`**).
- **Contributors vs release publisher (`pom.xml`):** For normal commits and PRs, **do not** change **`<version>`**, **`<scm><tag>`**, or **`distributionManagement`**, and leave **`maven-gpg-plugin`** **`skip`** at **`true`** so **`mvn verify`** does not require a signing key. The release publisher follows **MAINTAINERS.md** (version bump, **`CHANGELOG.md`**, GPG, staging servers, tag) in a dedicated release process.
- **Dependencies:** Resolve from **Maven Central** only - see **`README.md`** and **`pom.xml`**. Runtime SOAP stack is Eclipse EE4J **`com.sun.xml.ws:jaxws-rt`** (see **`README.md`**).
- **CI:** **`.github/workflows/ci.yml`** - **`mvn -B -Dgpg.skip=true`** on **Ubuntu** (**`ubuntu-latest`**, **Temurin 11**): **`verify`** then **`sample`** profile compile (no **`cp settings.xml`**; same as a default local checkout).
- **Do not commit to git:** **`local.properties`**, a personal **`settings.xml`** at the repo root if it contains secrets or machine-only paths you do not intend to share, real keystores, passwords, API tokens, private keys, or environment-specific production HI URLs. See **`SECURITY.md`**.
- **Line endings:** the repo expects **LF** (see **`.gitattributes`**).
  - **Windows:** in this clone, **`git config core.autocrlf false`** before committing so you do not rewrite every line ending in a PR.
  - **macOS / Linux:** defaults are often **`input`** or **`false`**; keep LF unless your **global** Git config forces CRLF. In that case, align with **`.gitattributes`** or team policy before pushing.
- **Changes:** Keep commits focused; match existing style and **`pom.xml`** conventions.
- **Automation / build details:** **`AGENTS.md`**. WSDL/codegen and dependency notes for maintainers: **`MAINTAINERS.md`**.
